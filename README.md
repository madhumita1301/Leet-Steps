🧠 LeetSteps
Nudge your brain, not spoil the fun.

LeetSteps is a minimalistic and user-friendly Chrome Extension for LeetCode that provides subtle, level-based hints — helping users evolve their thought process from 
brute force to optimal solutions without revealing the full answer.

🚀 Features
- Step-by-step hints from brute-force to optimal
- Designed to guide, not solve — so you learn while doing
- Automatically detects which LeetCode problem you're on
- No distractions, clean UI
- Easy to install and use

✨ Future Ideas
 - AI-generated hints via ChatGPT API
 - User can submit their own hints
 - Toggle difficulty level of hints
 - Dark Mode UI
